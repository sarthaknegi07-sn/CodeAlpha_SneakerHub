# 👟 SneakerHub

A full-stack e-commerce web application for browsing and purchasing sneakers. Built using the MERN stack with a responsive user interface and secure backend APIs.

---

## 🚀 Features

- User Authentication (Login & Signup)
- Browse Sneakers
- Search Products
- Add to Cart
- Responsive UI
- Secure Backend APIs
- MongoDB Database Integration

---

## 🛠 Tech Stack

### Frontend
- React.js
- HTML
- CSS
- JavaScript
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sarthaknegi07-sn/CodeAlpha_SneakerHub.git
```

### Install Backend

```bash
cd Backend
npm install
```

### Install Frontend

```bash
cd ../Frontend
npm install
```

---

## ▶️ Run Project

### Backend

```bash
cd Backend
npm start
```

### Frontend

```bash
cd Frontend
npm start
```
